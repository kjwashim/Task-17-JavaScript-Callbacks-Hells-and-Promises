# Task-17-JavaScript-Callbacks-Hells-and-Promises


This project demonstrates the fundamentals of **JavaScript asynchronous programming** using **Callbacks**, **Callback Hell**, and **Promises**. It is created as part of the Upskill Mafia JavaScript assignment.

## Objective

The objective of this project is to understand how asynchronous JavaScript works by implementing:

* Callback Functions
* Callback Hell
* JavaScript Promises
* Error Handling with Promises

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
Task-17-JavaScript-Callbacks-Hells-and-Promises/
│── index.html
│── style.css
│── script.js
└── README.md
```

## Features

* Demonstrates JavaScript callback functions.
* Shows how nested callbacks create Callback Hell.
* Uses Promises for cleaner asynchronous code.
* Handles successful and failed Promise states.
* Displays appropriate error messages for invalid operations.
* Beginner-friendly project structure.

## Concepts Covered

* Callback Functions
* Callback Hell
* Promise
* `resolve()`
* `reject()`
* `.then()`
* `.catch()`
* Arrow Functions
* Asynchronous JavaScript

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/kjwashim/Task-17-JavaScript-Callbacks-Hells-and-Promises.git
```

2. Open the project folder.

```bash
cd Task-17-JavaScript-Callbacks-Hells-and-Promises
```

3. Open `index.html` in your browser.

4. Open the browser's **Developer Console** (F12 → Console) to view the output.
